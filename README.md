通过该视频，可以很快的掌握 pytorch 编程的基本架构 对简单的线性回归（一层全连接层），简单的深度神经网络，卷积神经网络，LSTM 的搭建有一定了解 在最后的实战板块，通过调用已训练模型并修改其全连接层参数，完成了模型的移植和使用
